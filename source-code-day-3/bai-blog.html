React Native là gì?

React Native là một trong những công nghệ đang chiếm thế thượng phong trong phát triển ứng dụng di động đa nền tảng, giúp các start-up và các công ty lớn nhanh chóng phát hành ứng dụng với chi phí và thời gian thấp hơn rất nhiều so với trước đây. Vậy:
React Native tốt không?
Có!
Bạn có muốn học nó không?
Tất nhiên có!

Mình cũng vậy, mình đã tìm hiểu về React Native và cách nó giúp mình xây dựng các ứng dụng iOS và Android.

Câu hỏi đặt ra là Khi nào thì nên bắt đầu học React Native? Theo mình, trước khi "dấn thân" sang lĩnh vực di động này, bạn cần nắm chắc kiến thức về lập trình web đã, ít nhất là HTML, CSS và Javascript. Thành thật mà nói thì lập trình ứng dụng di động không phải là món dễ ăn đâu, nó khó hơn lập trình web đó, và React Native cũng vậy. Tuy nhiên, vì React Native được viết bằng Javascript nên nếu đã thành thạo Javascript, bạn hoàn toàn có thể chinh phục được công nghệ này 🤗
